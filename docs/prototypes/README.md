# Prototypes Folder

This folder contains all prototype evidence for Lab 2 requirements.

## Structure

Each team member should create a subfolder with their prototypes:

```
prototypes/
├── quy/
│   ├── prototype1_before.png
│   ├── prototype1_after.png
│   ├── prototype2_before.png
│   ├── prototype2_after.png
│   └── changelog.md
├── minh/
├── cap/
├── duc/
├── kiet/
└── khang/
```

## Requirements

Each team member must provide:
- **2 prototypes** (before and after versions based on peer interview feedback)
- **Change log** documenting:
  - What feedback was received
  - From whom (peer interview)
  - What changes were made in response
  - Rationale for changes

## Prototype Change Log Template

See `prototype_changelog_template.md` for the standard format to document changes.
